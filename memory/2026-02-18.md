# 2026-02-18 Daily Log

## Morning/Afternoon Tasks & Issues

*   **PDF 텍스트 추출 지연:**
    *   사용자가 `.hwp` 문서가 포함된 PDF 파일 및 이미지 기반 PDF 파일을 제공함.
    *   `read` 툴과 `web_fetch` 툴로는 PDF에서 텍스트 추출에 실패함.
    *   명령줄 `pdftotext` 도구를 사용하기로 결정하고 설치를 시도함.
    *   `pdftotext`가 기본 설치되어 있지 않음을 확인.
    *   사용자가 Chocolatey 설치를 완료했다고 알림.
    *   `choco.exe`의 경로가 PATH에 없어 실행 불가. 사용자가 경로를 `C:\ProgramData\chocolatey\bin\choco.exe`로 알려줌.
    *   해당 경로로 `choco.exe`를 실행했으나, `poppler` (pdftotext 포함)가 이미 설치되어 있다고 나옴.
    *   **현재 상태:** `pdftotext.exe`가 시스템에 설치는 되어 있으나, 정확한 경로를 알지 못해 호출 불가. 사용자가 `pdftotext.exe`의 경로를 찾아 알려주기로 함.
    *   `.NET Framework 4.5` 이상 버전 설치 여부 확인 결과, `Release` 값 `533509`로 요구사항 충족 확인. (버전 문제는 아님)

*   **`sessions_spawn` 권한 오류:**
    *   `watsonbot` 서브 에이전트 스폰 시 `agentId is not allowed for sessions_spawn` 오류 발생.
    *   `AGENTS.md` 파일에 서브 에이전트 스폰 허용 설정이 명시되지 않아 발생한 것으로 추정.
    *   현재 OpenClaw 시스템 정책상 직접 해결 불가능.

*   **NK 세포치료제 RFP 공동과제 분석 (보류):**
    *   `왓슨봇`에게 할당하려 했으나 `sessions_spawn` 오류로 불가.
    *   코어봇이 직접 분석을 진행하기로 했으나, PDF 텍스트 추출 문제로 인해 추가 자료가 없어 본격적인 분석 보류 중.

## Decisions

*   `pdftotext.exe`의 정확한 경로를 확보하는 것이 현재 최우선 과제.
*   `sessions_spawn` 권한 문제는 시스템 설정 문제로 현재로서는 우회 불가.
*   NK 세포치료제 RFP 분석은 `pdftotext` 경로 확보 후 PDF 자료 분석이 가능해지면 본격적으로 진행.
